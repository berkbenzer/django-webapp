Install Kubernetes Cluster to AWS

  • Execute the commands below to Install EKS service.
    • 
git clone https://github.com/hashicorp/learn-terraform-provision-eks-cluster
cd learn-terraform-provision-eks-cluster
terraform init
terraform plan

  • I will not share entire plan but if you run without any changes it should be out put the same.
    • 
Plan: 67 to add, 0 to change, 0 to destroy.

  • Output of terraform apply command. Enter "yes" or run the command terraform apply --auto-approve
  • It will take 10 or max 15 minute to create EKS infrastructure. Login to AWS console and navigate to the Region that you install the cluster.
    •
