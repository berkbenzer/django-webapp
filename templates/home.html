Purpose of the This Web Page

    I moved to Germany 2022 as an IT professional while I thought I could make a living with only English, I realized that things were not that easy. (Anyway, I figured it out before it was too late :D) Then I started practice what is this German language everybody talks about "uhh Artikles die,der,das is hard to memorize" I was not paying attention that much. But after I realized that the entire structure of the Language is depends on the "die, der ,das". I felt "okay it will not be that easy road" First, I thought it is better to develope my own programme the way I want, and now I practice with this programme for daily word and article exercises.

    You can view and add more functionality to this application on Github link.

    If you run the application on your mobile or another platform that supports Python it works properly but for the web version I will continue to develop.(currently it is bit bugy :D)

    I started to practice with this application on my mobile and used in my spare time or while traveling. Afterwards, I decided to convert this application into a Web App one by one. Since I was developing with Python, it would be easier to use Django. Because I am not a Software Developer, I work as a Database and Linux Engineer. Since I wanted to share what kind of infrastructure I used when converting this application to Web App, I created this web page not only for German Language Practice also as a resource where we can share technically.

    Okay I talked a lot lets dive in.


    Technology stacks to be used: AWS, Foreman, Linux, Kubernetes, Docker, Docker Hub, Github, Jenkins, Nginx, Django, Terraform, Ansible.

    AWS: It will be use for the Server side

    Foreman: It will be used for Centralized Patch Managemant System for our Server Farm

    Linux: Not a big fan of Windows :D

    Kubernetes It will be used for to manage containers

    Docker Our application will work on Docker Containers

    Docker Hub We Will keep our Docker Images in Docker Hub

    Github We Will store our codes

    Jenkins We will use for CI process

    Nginx We will use for Reverse Proxy

    Terraform Provisioning for our Infrastructure

    Ansible It will help us for Configuration Managemant
