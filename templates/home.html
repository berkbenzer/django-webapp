Purpose of the This Web Page

    Greetings and welcome! I'm Berk, I moved to Germany 2022 as an IT professional while I thought I could make a living with only English, I realized that things were not that easy. (Anyway, I figured it out before it was too late lol). General opinionis that German is a very diffucult language but I was not paying attention. And then I got that important slap in the face from der, die , das. I was thinking it just a cliche people talk but actually when you start to learn German you realized that the entire structure of the Language is depends on the "der, die ,das". I felt "okay it will not be that easy road" First, I thought it is better to develope my own programme the way I want, and now I practice with this programme for daily exercises.


    Afterwards, I decided to convert this application into a Web App one by one. Since I was developing with Python, it would be easier to use Django and then I can share the content what needs to be done at the Infra view to publish a Web App. I am planing to turn this Web App to Technical Dictionary and maybe improve the German Practice section.


    You can view and add more functionality to this application on my Github link.


    If you run the application on your mobile(you need terminal for to run) or another platform that supports Python it works properly but for the web version I will continue to develop.


    Okay I already talked a lot, lets dive in.
